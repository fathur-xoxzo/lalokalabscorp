# Introduction

Custom WP theme for Laloka Labs corporate site

## Getting started

- Install Wordpress
- Clone theme into `wp-content/themes` directory
- Install plugins
- Import settings

## Plugins

- Advanced Custom Fields
- Classic Editor
- Classic Widgets
- Custom Post Type UI
- Duplicate Page
- Easy Add Thumbnail
- Forminator
- Loco Translate
- Polylang
- Post Types Order
